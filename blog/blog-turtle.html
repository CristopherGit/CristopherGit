COMO CUIDAR A TU TORTUGA

1.Vivienda

Terrario o acuario: Proporciona un espacio adecuado para tu tortuga. Asegúrate de que sea lo suficientemente grande para permitir que se mueva con facilidad Luz y calor: Las tortugas necesitan acceso a luz solar o a lámparas UVB para la síntesis de vitamina D3. También requieren una fuente de calor para mantener una temperatura adecuada

2.Alimentacion

Proporciona una dieta equilibrada que incluya pellets de alta calidad diseñados para tortugas acuáticas, así como alimentos frescos como verduras, frutas y proteínas como insectos o gusanos No alimentes en exceso y e6ita alimentos tóxicos para las tortugas, como ciertos tipos de lechuga

3.Veterinario

Llega a tu tortuga a revisiones veterinarias regulares para asegurarte de que esté sana y para pre6enir posibles problemas de salud.
